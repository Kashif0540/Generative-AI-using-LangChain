# 1. What is Artificial Intelligence?

Artificial Intelligence (AI) is a branch of computer science that focuses on building machines capable of performing tasks that normally require human intelligence.

These tasks include:

- Learning
- Reasoning
- Problem Solving
- Decision Making
- Language Understanding
- Image Recognition

AI is an umbrella field that contains many specialized areas.

---

## Why is AI Important?

- Automates repetitive tasks
- Improves decision making
- Saves time and cost
- Enables intelligent applications
- Powers modern digital products

---

## Real-World Examples

- Google Search
- Google Maps
- Netflix Recommendations
- Self-driving Cars
- Virtual Assistants

---

# 2. Machine Learning

Machine Learning (ML) is a subset of Artificial Intelligence.

Instead of explicitly programming every rule, machines learn patterns from data.

Traditional Programming:

Input + Rules → Output

Machine Learning:

Input + Output → Machine learns Rules

---

## Types of Machine Learning

- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning

---

## Why Machine Learning Matters

Almost every modern AI application depends on Machine Learning.

Examples:

- Spam Detection
- Fraud Detection
- House Price Prediction
- Recommendation Systems

---

# 3. Deep Learning

Deep Learning is a specialized branch of Machine Learning.

It uses Artificial Neural Networks inspired by the human brain.

Deep Learning performs exceptionally well on:

- Images
- Audio
- Videos
- Natural Language

---

## Applications

- Face Recognition
- Speech Recognition
- Autonomous Driving
- Medical Imaging

---

# 4. What is Generative AI?

Generative AI is a branch of AI that creates new content instead of only analyzing existing data.

Generated content may include:

- Text
- Images
- Videos
- Music
- Code
- Audio

Unlike traditional AI, Generative AI produces original outputs.

---

## Examples

- ChatGPT
- Claude
- Gemini
- GitHub Copilot
- Midjourney
- Stable Diffusion

---

# 5. AI → ML → DL → GenAI Relationship

Artificial Intelligence

Generative AI

Every Generative AI model is built using Deep Learning techniques.

# 6. Why is Generative AI Growing So Fast?

Several factors contributed to the rapid growth of Generative AI.

- Large datasets
- Powerful GPUs
- Better algorithms
- Cloud computing
- Transformer Architecture
- Large Language Models
