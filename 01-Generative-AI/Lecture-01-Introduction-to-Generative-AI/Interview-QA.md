# 🎤 Interview Questions & Answers — Lecture 01

> These questions focus on conceptual understanding rather than memorization.

---

# Basic Level

## Q1. What is Artificial Intelligence?

### Answer

Artificial Intelligence (AI) is a branch of computer science that develops systems capable of performing tasks that typically require human intelligence, such as learning, reasoning, problem-solving, decision-making, and language understanding.

---

## Q2. What is Machine Learning?

### Answer

Machine Learning (ML) is a subset of AI where algorithms learn patterns from data instead of being explicitly programmed. The model improves its performance as it is exposed to more data.

---

## Q3. What is Deep Learning?

### Answer

Deep Learning is a specialized branch of Machine Learning that uses multi-layer artificial neural networks to automatically learn complex patterns from large datasets.

---

## Q4. What is Generative AI?

### Answer

Generative AI is a type of Artificial Intelligence that creates new content such as text, images, code, audio, or videos based on patterns learned from existing data.

---

## Q5. Give examples of Generative AI applications.

### Answer

Examples include:

# Intermediate Level

## Q6. Explain the relationship between AI, Machine Learning, Deep Learning, and Generative AI.

### Answer

Artificial Intelligence is the broad field.

Machine Learning is a subset of AI.

Deep Learning is a subset of Machine Learning.

Generative AI is primarily built using Deep Learning models.

Relationship:

AI → Machine Learning → Deep Learning → Generative AI

---

## Q7. Why is Generative AI becoming so popular?

### Answer

Major reasons include:

- Availability of large datasets
- Advances in GPU computing
- Transformer architecture
- Large Language Models (LLMs)
- Cloud computing infrastructure

---

## Q8. What skills are required to become an AI Engineer?

### Answer

An AI Engineer should understand:

- Python programming
- Mathematics
- Machine Learning
- Deep Learning
- Transformers
- Large Language Models
- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- AI Agents
- Deployment and MLOps fundamentals

---

## Q9. Why are projects important while learning AI?

### Answer

Projects help demonstrate practical knowledge, improve problem-solving skills, strengthen concepts, and build a professional portfolio for internships and jobs.

---

## Q10. Why should developers use Git and GitHub?

### Answer

Git enables version control by tracking code changes locally, while GitHub hosts repositories online, making collaboration, backup, and portfolio management easier.

---

# Scenario-Based Questions

## Q11. You have completed several AI courses but have not built any projects. What challenge might you face during interviews?

### Answer

Interviewers usually evaluate practical problem-solving ability. Without projects, it becomes difficult to demonstrate how theoretical knowledge can be applied to real-world situations.

---

## Q12. Why should concepts be learned before frameworks?

### Answer

Frameworks evolve quickly, but concepts remain applicable across technologies. A strong conceptual foundation makes learning new tools significantly easier.

---

## Q13. If a new AI framework becomes popular tomorrow, how would you learn it efficiently?

### Answer

A strong understanding of Python, Machine Learning, Deep Learning, and Transformer architecture allows developers to adapt quickly to new frameworks by focusing only on framework-specific syntax and workflows.

# Interview Tips

✔ Explain concepts in your own words.

✔ Use real-world examples whenever possible.

✔ Keep answers structured and concise.

✔ Differentiate related concepts clearly (AI vs ML vs DL vs GenAI).

✔ Demonstrate curiosity and a continuous learning mindset.

---

# Key Takeaway

Interviewers rarely expect beginners to know everything. They expect a clear understanding of the fundamentals, logical thinking, and evidence that you are actively learning through projects and documentation.
